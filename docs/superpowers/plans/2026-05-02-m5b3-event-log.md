# M5.b.3: CRC16 + event_log Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Scan-on-boot ring buffer for event records. After this milestone the firmware can append a 32-byte event record at any time; appended records survive a reboot; the scan finds the most-recent valid record and resumes appending after it. Spec § 6 calls this `event_log` — 64 sectors at `0x004000`–`0x043FFF`.

**Scope:** event_log only. session_log (M5.b.4) follows the same pattern with smaller capacity. `persist_task` queue (M5.b.5) wraps the synchronous append in a FreeRTOS queue. Crash-loop detection (M5.b.6) uses event_log to detect rapid resets.

**Architecture:**
- 256 KB region (64 × 4 KB sectors) of 32-byte records (128 per sector → 8192 total).
- **Scan-on-boot.** Walk all 8192 slots; classify each as `valid` (CRC16 OK), `blank` (all 0xFF — erased), or `corrupt` (CRC mismatch but not blank). Track the highest `(boot_count, timestamp)` pair across all valid records — that's the tail. Head = next slot after tail.
- **Append at head.** If head crosses a sector boundary, erase the new sector before programming. Synchronous-blocking; called from task context (or main pre-scheduler for tests).
- **CRC16-CCITT-FALSE** (polynomial 0x1021, init 0xFFFF, no final XOR). Spec calls for "CRC16" without specifying flavour; CCITT-FALSE is the standard for short embedded records and matches the byte-loop that fits comfortably in flash.
- No header sector — head is reconstructed every boot. ~256 KB of W25Q reads × ~70 ns/byte at 15 MHz SPI = ~18 ms per scan. Acceptable; runs once at boot before scheduler.
- **Boot count comes from M5's `boot_count_increment()`.** Caller stamps it via `event_log_set_boot_count(u16)` after the increment. event_log uses it to break ties between same-timestamp records and to identify "this boot's records vs prior boots".

**Tech Stack:** Pingpong helper (M5.b.1) is unused here — event_log has different shape (ring, not 2-slot). Reuses W25Q HAL (M4) directly.

**Hardware preconditions:**
- M5.b.2 tagged. Calibration cache live.
- Bench unit on USB 5 V is sufficient.

**Success criteria:**
1. First boot after flashing (event_log empty): `event_log: scan complete, head=0x004000 slot=0 (empty)`.
2. Three programmatic appends from main at boot — scan reports `head=0x004000 slot=3 (after 3 valid)`.
3. Reset: scan finds the 3 prior records, head = slot 3 still.
4. After 4th append: head = slot 4.
5. After enough appends to cross a sector boundary (slot 128): scan reports `head=0x005000 slot=0`, sector 1 was erased before the boundary write.
6. PD4 still blinks; uptime ≥ 5 min, no resets.
7. Build: text < 20 KB.

/* CRC16-CCITT-FALSE: polynomial 0x1021, init 0xFFFF, no final XOR.
 * Used for the 32-byte event_log / session_log records (CRC32 is

- [ ] **Step 2: Create `src/persist/crc16.c`**

```c
#include "crc16.h"

uint16_t crc16_ccitt(const void *data, size_t len)
{
    const uint8_t *p = (const uint8_t *)data;
    uint16_t crc = 0xFFFFu;
    while (len--) {
        crc ^= ((uint16_t)*p++) << 8;
        for (unsigned i = 0; i < 8; ++i) {
            crc = (crc & 0x8000u) ? (uint16_t)((crc << 1) ^ 0x1021u)
                                  : (uint16_t)(crc << 1);
        }
    }
    return crc;
}
```

```c
#ifndef OPENBHZD_PERSIST_EVENT_LOG_H
#define OPENBHZD_PERSIST_EVENT_LOG_H

#include <stddef.h>
#include <stdint.h>

#define EVENT_LOG_BASE          0x004000U
#define EVENT_LOG_SECTORS       64U
#define EVENT_LOG_SECTOR_SIZE   4096U
#define EVENT_LOG_REGION_SIZE   (EVENT_LOG_SECTORS * EVENT_LOG_SECTOR_SIZE)
#define EVENT_LOG_RECORD_SIZE   32U
#define EVENT_LOG_RECORDS_PER_SECTOR  (EVENT_LOG_SECTOR_SIZE / EVENT_LOG_RECORD_SIZE)
#define EVENT_LOG_TOTAL_RECORDS (EVENT_LOG_SECTORS * EVENT_LOG_RECORDS_PER_SECTOR)

/* 32 bytes total. Layout per spec § 6. CRC16 covers bytes 0..29. */
struct __attribute__((packed)) event_record {
    uint32_t timestamp;
    uint16_t boot_count;
    uint16_t fault_id;
    uint8_t  j1772_state;
    uint8_t  evse_state;
    int16_t  cp_mv;
    uint16_t cc_amps;
    uint16_t ntc1_dC;
    uint16_t ntc2_dC;
    uint16_t active_amps_x10;
    uint8_t  reserved[10];
    uint16_t crc16;
};
_Static_assert(sizeof(struct event_record) == 32, "event_record must be 32 B");

/* Scan the entire log to find the head (next-write slot). Logs the
 * scan result. Returns 0 always (a corrupt log isn't fatal — we just
 * advance the head past corruption). Idempotent; safe to call at boot. */
int event_log_init(void);

/* Caller (main) supplies the current boot_count after M5's
 * boot_count_increment() runs. Used by event_log_append() to stamp
 * each record. */
void event_log_set_boot_count(uint16_t boot_count);

/* Append a record. Caller fills all fields EXCEPT boot_count and crc16
 * — the helper sets boot_count from event_log_set_boot_count() and
 * computes crc16. Returns 0 on success, <0 on W25Q error. */
int event_log_append(struct event_record *rec);

/* Debug: read the n-th record from the start of the log into out_rec.
 * Returns 0 on valid, 1 on blank slot, <0 on read error / out-of-range.
 * out_rec is populated even on rc==1 (will be all 0xFF). */
int event_log_read_nth(uint32_t n, struct event_record *out_rec);

/* Debug: how many slots from the start of the log to the current head. */
uint32_t event_log_head_slot(void);

#endif
```

- [ ] **Step 2: Create `src/persist/event_log.c`**

```c
#include "event_log.h"
#include "crc16.h"
#include "../hal/w25q.h"
#include "../hal/uart.h"
#include <string.h>

static uint32_t s_head_slot = 0;        /* 0..EVENT_LOG_TOTAL_RECORDS */
static uint16_t s_boot_count = 0;
static int      s_initialized = 0;

static uint32_t slot_to_addr(uint32_t slot)
{
    return EVENT_LOG_BASE + slot * EVENT_LOG_RECORD_SIZE;
}

/* Returns: 1=valid, 0=blank, -1=corrupt (CRC fail, non-blank). */
static int classify_record(const struct event_record *rec)
{
    const uint8_t *b = (const uint8_t *)rec;
    int all_ff = 1;
    for (size_t i = 0; i < sizeof *rec; ++i) {
        if (b[i] != 0xFFu) { all_ff = 0; break; }
    }
    if (all_ff) return 0;

    uint16_t expected = crc16_ccitt(rec, sizeof *rec - 2U);
    if (rec->crc16 == expected) return 1;
    return -1;
}

int event_log_init(void)
{
    uint32_t valid = 0, corrupt = 0, blank = 0;
    uint32_t latest_boot = 0;
    uint32_t latest_ts   = 0;
    uint32_t latest_slot = 0;
    int      seen_valid  = 0;

    /* Read one sector (4 KB, 128 records) at a time to amortise
     * SPI read overhead. */
    static uint8_t sector_buf[EVENT_LOG_SECTOR_SIZE];

    for (uint32_t s = 0; s < EVENT_LOG_SECTORS; ++s) {
        w25q_read(EVENT_LOG_BASE + s * EVENT_LOG_SECTOR_SIZE,
                  sector_buf, EVENT_LOG_SECTOR_SIZE);

        for (uint32_t r = 0; r < EVENT_LOG_RECORDS_PER_SECTOR; ++r) {
            const struct event_record *rec =
                (const struct event_record *)(sector_buf + r * EVENT_LOG_RECORD_SIZE);
            int c = classify_record(rec);
            if (c == 0) { ++blank; continue; }
            if (c < 0) { ++corrupt; continue; }
            ++valid;

            uint32_t key_boot = rec->boot_count;
            uint32_t key_ts   = rec->timestamp;
            uint32_t slot     = s * EVENT_LOG_RECORDS_PER_SECTOR + r;
            if (!seen_valid ||
                key_boot > latest_boot ||
                (key_boot == latest_boot && key_ts >= latest_ts)) {
                latest_boot = key_boot;
                latest_ts   = key_ts;
                latest_slot = slot;
                seen_valid  = 1;
            }
        }
    }

    if (seen_valid) {
        s_head_slot = (latest_slot + 1U) % EVENT_LOG_TOTAL_RECORDS;
    } else {
        s_head_slot = 0;
    }

    printk("event_log: scan complete: valid=%u corrupt=%u blank=%u head=0x%06x slot=%u\n",
           (unsigned)valid, (unsigned)corrupt, (unsigned)blank,
           (unsigned)slot_to_addr(s_head_slot), (unsigned)s_head_slot);

    s_initialized = 1;
    return 0;
}

void event_log_set_boot_count(uint16_t boot_count)
{
    s_boot_count = boot_count;
}

int event_log_append(struct event_record *rec)
{
    if (!s_initialized) return -1;
    if (rec == NULL) return -1;

    rec->boot_count = s_boot_count;
    rec->crc16 = crc16_ccitt(rec, sizeof *rec - 2U);

    uint32_t addr = slot_to_addr(s_head_slot);

    /* If we're about to write to slot 0 of a sector, erase it first.
     * Slot 0 = sector boundary. Only erase if the sector contains any
     * non-blank bytes (avoid wearing freshly-erased sectors). */
    if (s_head_slot % EVENT_LOG_RECORDS_PER_SECTOR == 0U) {
        /* Quick blank check on the first record of the sector. */
        struct event_record probe;
        w25q_read(addr, &probe, sizeof probe);
        const uint8_t *b = (const uint8_t *)&probe;
        int needs_erase = 0;
        for (size_t i = 0; i < sizeof probe; ++i) {
            if (b[i] != 0xFFu) { needs_erase = 1; break; }
        }
        if (needs_erase) {
            if (w25q_erase_sector(addr) != 0) return -2;
        }
    }

    if (w25q_program(addr, rec, sizeof *rec) != 0) return -3;

    s_head_slot = (s_head_slot + 1U) % EVENT_LOG_TOTAL_RECORDS;
    return 0;
}

int event_log_read_nth(uint32_t n, struct event_record *out_rec)
{
    if (n >= EVENT_LOG_TOTAL_RECORDS || out_rec == NULL) return -1;
    w25q_read(slot_to_addr(n), out_rec, sizeof *out_rec);
    return classify_record(out_rec) >= 0 ? 0 : 1;
}

uint32_t event_log_head_slot(void)
{
    return s_head_slot;
}
```

## Self-review

**Spec coverage (M5.b.3 slice):**

| Spec § 6 requirement | This plan |
|---|---|
| event_log: 64 sectors × 128 records | Tasks 1–3 (`EVENT_LOG_*` constants in header) |
| 32-byte record per spec layout | Task 2 (`struct event_record`) |
| Scan-on-boot head discovery (no header sector) | Task 2 (`event_log_init`) |
| Append-only ring; erase next sector on wrap | Task 2 (`event_log_append`) |
| All-0xFF = empty | Task 2 (`classify_record`) |
| CRC16 (vs CRC32 for ping-pong records) | Task 1 |
| session_log | DEFERRED to M5.b.4 |
| persist_task queue | DEFERRED to M5.b.5 |

**Placeholders:** none.

**Type/name consistency:** `event_log_*` matches the `boot_config_*` / `calibration_*` shape. `crc16_ccitt` (not `crc16` or `crc16_kermit` etc.) makes the polynomial choice obvious at the call site.

**Wear:** 32 records/charging-session × 10 sessions/day = 320 records/day. 8192 records ÷ 320 = 25.6 days per ring lap. Each lap erases all 64 sectors once → 365/25.6 × 64 ≈ 14.3 sector erases per sector per year. 100K cycles ÷ 14.3 = 7,000 years. Negligible.

**Static buffer cost:** `sector_buf[4096]` in event_log_init. 4 KB of bss. Acceptable; init is one-shot.
