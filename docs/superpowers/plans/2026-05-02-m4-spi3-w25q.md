# M4: SPI3 + W25Q64 Driver Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Bring up the W25Q64 SPI flash chain over hardware SPI3 (PB3 SCK / PB4 MISO / PB5 MOSI + PB6 GPIO chip-select). Validate by reading the JEDEC ID (must match `0xEF4017` for W25Q64JV) and round-tripping a 4 KB sector — erase, page-program a known 0xA5/0x5A pattern, read back, compare byte-by-byte. M4 ships only the HAL; the persistence layer (ping-pong, event_log, session_log) is M5.

**Architecture:** SPI3 chip-name = SPI2 in GD32 SPL nomenclature. APB1 = 30 MHz at our 120 MHz HCLK; with prescaler /2 we get **15 MHz SPI clock**, well within W25Q64JV's 80 MHz spec and the spec's "12–25 MHz" guidance. CS is GPIO-bit-banged via BSRR/BRR around each transaction. All SPI ops are blocking, polling-driven, no DMA — matches the spec's "synchronous blocking ops only". One file owns the peripheral (`hal/spi3.c`); a second file (`hal/w25q.c`) builds the W25Q command vocabulary on top. `safety_task` does NOT touch SPI3 — only `persist_task` will (in M5). For M4, validation runs from a one-shot test invoked from `main()` after init; the result prints via printk, then normal task execution proceeds.

**Tech Stack:** GD32 SPL `spi`, existing FreeRTOS / UART / GPIO from M0–M3.

**Hardware preconditions:**
- M3 complete and tagged `m3-cp-pwm-state-machine`. Build/flash/monitor toolchain functional.
- W25Q64 chip populated on the bench unit (already wire-traced in M2; SPI3 pads confirmed at PB3/4/5 + PB6 CS).
- Bench unit on USB 5 V is sufficient — SPI flash is on the 3.3 V rail, not 12 V.
- Recovery backup (`stock-mcu-V1.0.066.bin`) on disk. **No backup of W25Q contents is required** for M4 — we only round-trip a previously-unused sector, but record this:
  - Round-trip target sector: **0x07F000** (the very last 4 KB sector of the 8 MB chip). Stock firmware's persistence sits in the lower addresses (`0x000000`–`0x04CFFF` per spec); the upper ~7.7 MB is "reserved (future)" and shouldn't contain any data we care about. Using the last sector keeps the test maximally far from any stock-firmware-relevant region.

**Success criterion:**
1. `JEDEC ID = 0xEF4017` printed at boot (or whatever the chip reports — Winbond W25Q64JV expected, but other 8 MB SPI NORs may have different IDs and that's fine as long as it parses cleanly and is non-zero).
2. Sector erase at 0x07F000 returns within ~50 ms.
3. Page program (256 bytes) at 0x07F000 with a 0xA5/0x5A alternating pattern returns within ~3 ms.
4. Read-back of the same 256 bytes matches byte-for-byte.
5. printk reports `W25Q round-trip PASS: 256 bytes match` (or FAIL with the first mismatch index + values).
6. PD4 still blinks at 1 Hz; uptime ≥ 5 min, no resets.

**Why each file is separate:**
- `hal/spi3.c` is the peripheral driver: configures CTL0/CTL1, enables clocks, exposes a single primitive `spi3_xfer(uint8_t)` that writes one byte and returns the simultaneously-received byte. Nothing W25Q-specific.
- `hal/w25q.c` builds the W25Q vocabulary (JEDEC ID, status read, WREN, sector erase, page program, read) on top of `spi3_xfer()` + `cs_assert/deassert()`. Could be replaced wholesale if we ever swap to a different SPI flash without touching `spi3.c`.

## SPI3 (= SPL SPI2) configuration

| Field | Value |
|---|---|
| Bus | APB1 (30 MHz) |
| Prescaler | /2 → 15 MHz SCK |
| Mode | Master, full-duplex |
| Frame | 8-bit, MSB first |
| Clock polarity / phase | CPOL=0, CPHA=0 (SPI mode 0 — W25Q standard) |
| NSS | Software-managed (we drive PB6 GPIO directly; NSS HIGH internally so peripheral is happy) |

GPIO already configured by `gpio_init_all()` from M2:
- PB3 SCK → AF push-pull 50 MHz
- PB4 MISO → input floating
- PB5 MOSI → AF push-pull 50 MHz
- PB6 CS → GPIO push-pull 50 MHz, idle HIGH

**SPI3 default-pins note:** PB3/4/5 are the *default* (non-remapped) pins for SPI3. **No AFIO remap needed** — SPL `spi_init()` works as-is.

---

## W25Q64 commands used in M4

| Mnemonic | Opcode | Notes |
|---|---:|---|
| READ_JEDEC_ID | 0x9F | Returns 3 bytes: manufacturer / type / capacity. W25Q64JV = 0xEF / 0x40 / 0x17 |
| READ_STATUS_1 | 0x05 | Returns 1 byte; bit 0 (BUSY) = 1 while erase/program in progress |
| WRITE_ENABLE | 0x06 | No data; sets WEL bit, must precede every erase/program |
| READ_DATA | 0x03 | Address = 24-bit big-endian. Slow-read (max ~50 MHz on JV); fine for our 15 MHz |
| SECTOR_ERASE_4KB | 0x20 | Address = 24-bit; erases 4 KB sector, takes 30–400 ms (typ ~50 ms) |
| PAGE_PROGRAM | 0x02 | Address + up to 256 bytes; takes 0.4–3 ms |

Polling pattern for erase/program: after issuing the command, send READ_STATUS_1 and loop reading the status byte until BUSY (bit 0) clears. Hard timeout: 500 ms for erase, 10 ms for program.

```c
#ifndef OPENEVCHARGER_HAL_SPI3_H
#define OPENEVCHARGER_HAL_SPI3_H

#include <stdint.h>

/* Initialise SPI3 (= SPL SPI2) at 15 MHz, mode 0, 8-bit MSB-first.
 * GPIO pads (PB3/4/5/6) must already be configured by gpio_init_all().
 * Idempotent. After this returns, spi3_xfer() is usable. */
void spi3_init(void);

/* Synchronous full-duplex byte exchange. Drives MOSI = `tx`, returns
 * whatever MISO clocks back during the same 8 SCK pulses. CS handling
 * is the caller's responsibility — see spi3_cs_assert/deassert. */
uint8_t spi3_xfer(uint8_t tx);

/* Drive PB6 LOW → CS asserted (W25Q listening). */
void spi3_cs_assert(void);

/* Drive PB6 HIGH → CS deasserted. Always call after a transaction
 * even if the chip didn't ack, otherwise the chip stays in its
 * current command state and the next transaction misbehaves. */
void spi3_cs_deassert(void);

#endif
```

- [ ] **Step 2: Create `src/hal/spi3.c`**

```c
#include "spi3.h"
#include "core/pin_map.h"
#include "gd32f20x.h"

void spi3_init(void)
{
    rcu_periph_clock_enable(RCU_SPI2);

    spi_parameter_struct cfg;
    spi_struct_para_init(&cfg);
    cfg.trans_mode           = SPI_TRANSMODE_FULLDUPLEX;
    cfg.device_mode          = SPI_MASTER;
    cfg.frame_size           = SPI_FRAMESIZE_8BIT;
    cfg.clock_polarity_phase = SPI_CK_PL_LOW_PH_1EDGE;   /* mode 0 */
    cfg.nss                  = SPI_NSS_SOFT;
    cfg.prescale             = SPI_PSC_2;                /* APB1/2 = 15 MHz */
    cfg.endian               = SPI_ENDIAN_MSB;
    spi_init(SPI2, &cfg);

    /* Software NSS: drive internal NSS HIGH so the peripheral doesn't
     * think it's been deselected. We use PB6 GPIO for the actual CS. */
    spi_nss_internal_high(SPI2);
    spi_enable(SPI2);
}

uint8_t spi3_xfer(uint8_t tx)
{
    /* Wait for TX buffer empty, write byte, wait for RX buffer non-empty,
     * read the simultaneously-clocked byte. This is the standard
     * synchronous-polling pattern. */
    while (RESET == spi_i2s_flag_get(SPI2, SPI_FLAG_TBE)) { }
    spi_i2s_data_transmit(SPI2, (uint16_t)tx);
    while (RESET == spi_i2s_flag_get(SPI2, SPI_FLAG_RBNE)) { }
    return (uint8_t)spi_i2s_data_receive(SPI2);
}

void spi3_cs_assert(void)
{
    gpio_bit_reset(PIN_W25Q_CS_PORT, PIN_W25Q_CS_PIN);
}

void spi3_cs_deassert(void)
{
    gpio_bit_set(PIN_W25Q_CS_PORT, PIN_W25Q_CS_PIN);
}
```

```c
#ifndef OPENEVCHARGER_HAL_W25Q_H
#define OPENEVCHARGER_HAL_W25Q_H

#include <stddef.h>
#include <stdint.h>

#define W25Q_PAGE_SIZE       256U
#define W25Q_SECTOR_SIZE    4096U

/* Initialise W25Q. Reads + caches JEDEC ID. SPI3 must already be up.
 * Returns 0 on success, -1 if JEDEC reads as all-zero or all-one
 * (indicates wiring/power problem). */
int w25q_init(void);

/* JEDEC ID as { mfr<<16 | type<<8 | capacity }. 0 if init failed.
 * W25Q64JV = 0x00EF4017. */
uint32_t w25q_jedec_id(void);

/* Read `len` bytes starting at flash address `addr`. */
void w25q_read(uint32_t addr, void *buf, size_t len);

/* Erase the 4 KB sector containing `addr`. Blocks until done.
 * Returns 0 on success, -1 on timeout (>500 ms). */
int w25q_erase_sector(uint32_t addr);

/* Program up to 256 bytes at `addr`. Caller MUST ensure the buffer
 * does not cross a 256-byte page boundary; if it does the W25Q wraps
 * within the page rather than progressing. Blocks until done. */
int w25q_program(uint32_t addr, const void *buf, size_t len);

#endif
```

- [ ] **Step 2: Create `src/hal/w25q.c`**

```c
#include "w25q.h"
#include "spi3.h"

/* Standard SPI NOR command set */
#define W25Q_CMD_READ_JEDEC      0x9F
#define W25Q_CMD_READ_STATUS_1   0x05
#define W25Q_CMD_WRITE_ENABLE    0x06
#define W25Q_CMD_READ_DATA       0x03
#define W25Q_CMD_SECTOR_ERASE    0x20
#define W25Q_CMD_PAGE_PROGRAM    0x02

#define STATUS_BUSY              0x01

#define ERASE_TIMEOUT_LOOPS    5000000U   /* very rough; ~500 ms at our spi3_xfer rate */
#define PROGRAM_TIMEOUT_LOOPS   100000U

static uint32_t s_jedec_id = 0;

static uint8_t read_status1(void)
{
    spi3_cs_assert();
    spi3_xfer(W25Q_CMD_READ_STATUS_1);
    uint8_t st = spi3_xfer(0xFF);
    spi3_cs_deassert();
    return st;
}

static int wait_not_busy(uint32_t max_loops)
{
    while (max_loops--) {
        if (!(read_status1() & STATUS_BUSY)) return 0;
    }
    return -1;
}

static void write_enable(void)
{
    spi3_cs_assert();
    spi3_xfer(W25Q_CMD_WRITE_ENABLE);
    spi3_cs_deassert();
}

int w25q_init(void)
{
    spi3_cs_deassert();

    spi3_cs_assert();
    spi3_xfer(W25Q_CMD_READ_JEDEC);
    uint8_t mfr = spi3_xfer(0xFF);
    uint8_t typ = spi3_xfer(0xFF);
    uint8_t cap = spi3_xfer(0xFF);
    spi3_cs_deassert();

    s_jedec_id = ((uint32_t)mfr << 16) | ((uint32_t)typ << 8) | cap;

    if (s_jedec_id == 0x000000 || s_jedec_id == 0xFFFFFF) {
        s_jedec_id = 0;
        return -1;
    }
    return 0;
}

uint32_t w25q_jedec_id(void)
{
    return s_jedec_id;
}

void w25q_read(uint32_t addr, void *buf, size_t len)
{
    uint8_t *p = (uint8_t *)buf;
    spi3_cs_assert();
    spi3_xfer(W25Q_CMD_READ_DATA);
    spi3_xfer((uint8_t)(addr >> 16));
    spi3_xfer((uint8_t)(addr >>  8));
    spi3_xfer((uint8_t)(addr));
    while (len--) {
        *p++ = spi3_xfer(0xFF);
    }
    spi3_cs_deassert();
}

int w25q_erase_sector(uint32_t addr)
{
    write_enable();

    spi3_cs_assert();
    spi3_xfer(W25Q_CMD_SECTOR_ERASE);
    spi3_xfer((uint8_t)(addr >> 16));
    spi3_xfer((uint8_t)(addr >>  8));
    spi3_xfer((uint8_t)(addr));
    spi3_cs_deassert();

    return wait_not_busy(ERASE_TIMEOUT_LOOPS);
}

int w25q_program(uint32_t addr, const void *buf, size_t len)
{
    if (len == 0 || len > W25Q_PAGE_SIZE) return -1;

    write_enable();

    const uint8_t *p = (const uint8_t *)buf;
    spi3_cs_assert();
    spi3_xfer(W25Q_CMD_PAGE_PROGRAM);
    spi3_xfer((uint8_t)(addr >> 16));
    spi3_xfer((uint8_t)(addr >>  8));
    spi3_xfer((uint8_t)(addr));
    while (len--) {
        spi3_xfer(*p++);
    }
    spi3_cs_deassert();

    return wait_not_busy(PROGRAM_TIMEOUT_LOOPS);
}
```

### Task 3: Boot-time round-trip self-test

**Files:**
- Modify: `src/main.c`

The self-test runs once at boot, after all peripherals are up but before tasks start. It exercises every HAL function in the order persistence will use them, prints PASS/FAIL with diagnostic detail, and then proceeds to normal task execution regardless of result. The result is non-fatal in M4 — production firmware (M5+) will refuse to boot if the JEDEC ID is wrong, but for M4 we want the rest of the system to keep running so we can inspect.

- [ ] **Step 1: Add the self-test function in `main.c`**

```c
#include "hal/uart.h"
#include "hal/gpio.h"
#include "hal/adc_scan.h"
#include "hal/cp_pwm.h"
#include "hal/adc_inject.h"
#include "hal/spi3.h"
#include "hal/w25q.h"
...

/* M4 self-test: round-trip 256 bytes at the last sector of the 8 MB chip.
 * Stock firmware uses 0x000000–0x04CFFF per the spec § 6 memory map;
 * 0x07F000 is in the upper "reserved" area, untouched by stock fw. */
#define M4_TEST_ADDR  0x07F000U

static void m4_self_test(void)
{
    int rc;

    /* Phase 1: JEDEC ID */
    if (w25q_init() != 0) {
        printk("W25Q: JEDEC init FAIL (probably no SPI / wrong wiring)\n");
        return;
    }
    uint32_t id = w25q_jedec_id();
    printk("W25Q: JEDEC ID = 0x%06x", (unsigned)id);
    if (id == 0x00EF4017) printk(" (W25Q64JV)\n");
    else                  printk(" (unrecognised — non-Winbond or different capacity)\n");

    /* Phase 2: erase */
    rc = w25q_erase_sector(M4_TEST_ADDR);
    if (rc != 0) {
        printk("W25Q: erase FAIL (timeout)\n");
        return;
    }
    printk("W25Q: erased sector @ 0x%06x\n", M4_TEST_ADDR);

    /* Phase 3: program */
    uint8_t pat[W25Q_PAGE_SIZE];
    for (unsigned i = 0; i < W25Q_PAGE_SIZE; ++i) {
        pat[i] = (i & 1) ? 0x5A : 0xA5;
    }
    rc = w25q_program(M4_TEST_ADDR, pat, sizeof pat);
    if (rc != 0) {
        printk("W25Q: program FAIL (timeout)\n");
        return;
    }
    printk("W25Q: programmed %u bytes @ 0x%06x\n", (unsigned)sizeof pat, M4_TEST_ADDR);

    /* Phase 4: read-back + compare */
    uint8_t back[W25Q_PAGE_SIZE];
    w25q_read(M4_TEST_ADDR, back, sizeof back);

    int first_mismatch = -1;
    for (unsigned i = 0; i < sizeof back; ++i) {
        if (back[i] != pat[i]) { first_mismatch = (int)i; break; }
    }
    if (first_mismatch < 0) {
        printk("W25Q round-trip PASS: %u bytes match\n", (unsigned)sizeof back);
    } else {
        printk("W25Q round-trip FAIL: first mismatch idx=%d expect=0x%02x got=0x%02x\n",
               first_mismatch,
               (unsigned)pat[first_mismatch], (unsigned)back[first_mismatch]);
    }
}
```
